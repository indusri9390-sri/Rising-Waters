# Rising-Waters
Flood Prediction System using Machine Learning to forecast flood risk from historical meteorological data. Implements Decision Tree, Random Forest, KNN, and XGBoost, with the best model deployed in a Flask web application. Supports early warning, disaster response, and scalable cloud deployment.
## Conclusion

The **Rising Waters – Flood Prediction System** demonstrates how machine learning can be applied to predict flood occurrences using weather and rainfall-related data. The project follows a complete machine learning workflow, including data collection, exploratory data analysis, preprocessing, model building, evaluation, and deployment.

Multiple machine learning algorithms, including **Decision Tree, Random Forest, K-Nearest Neighbors (KNN), and XGBoost**, were trained and compared to identify the best-performing model. The selected model was then integrated into a **Flask-based web application**, allowing users to enter weather parameters and receive flood predictions through a simple and user-friendly interface.

This project showcases practical skills in **Python, data analysis, machine learning, model deployment, and web application development**, while demonstrating how technology can support flood risk assessment and informed decision-making.
## Project Output

The Flood Prediction System was successfully deployed using Flask.
