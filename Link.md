## Description

The **Rising Waters** project was developed using powerful Python-based tools and libraries for data processing, machine learning, visualization, and web deployment. These technologies enable efficient model development, data analysis, visualization, and deployment of the flood prediction application.

---

### Anaconda Navigator
A desktop graphical user interface (GUI) used to manage Python environments, install packages, and launch data science applications.

**Link:** https://www.anaconda.com/download

---

### PyCharm
An Integrated Development Environment (IDE) designed for Python programming with features such as intelligent code completion, debugging, and project management.

**Link:** https://www.jetbrains.com/pycharm/

---

### NumPy
A fundamental Python library for numerical computing that provides support for large multidimensional arrays, matrices, and mathematical operations.

**Link:** https://numpy.org/doc/stable/

---

### Pandas
A powerful library for data manipulation and analysis, offering flexible data structures such as DataFrames for handling structured datasets.

**Link:** https://pandas.pydata.org/docs/

---

### Scikit-learn
An open-source machine learning library that provides algorithms for classification, regression, clustering, model evaluation, and data preprocessing.

**Link:** https://scikit-learn.org/stable/

---

### Matplotlib
A comprehensive visualization library used to create static, animated, and interactive graphs and charts for data analysis.

**Link:** https://matplotlib.org/stable/

---

### Seaborn
A high-level statistical visualization library built on Matplotlib that simplifies the creation of attractive and informative charts.

**Link:** https://seaborn.pydata.org/

---

### Flask
A lightweight Python web framework used to build web applications and REST APIs for deploying the machine learning flood prediction model.

**Link:** https://flask.palletsprojects.com/

---

## Summary

These technologies together support the complete development lifecycle of the **Rising Waters** Flood Prediction System—from data collection and preprocessing to machine learning model development, visualization, and deployment through a web-based application.
